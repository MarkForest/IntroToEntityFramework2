Домашнее задание
Создайте Windows Forms приложение для работы с на-
шей БД с использованием Entity Framework по технологии
Database First. В главном окне приложения должен со-
держаться элемент TabControl с тремя вкладками: Books,
Authors и Publishers. Каждая вкладка должна обслужи-
вать одну из таблиц нашей БД и выполнять добавление,
удаление и редактирование записей в своей таблице.
Для добавления, редактирования и удаления записей
использовать формы. Для отображения результатов ме-
тодов GetAllAuthors(),GetAllBooks() и GetAllPublishers()
использовать DataGridView.
